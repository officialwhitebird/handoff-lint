# Security Policy

## Supported Versions

Only the current version (latest commit or release) of handoff-lint is supported for security updates.

## Reporting a Vulnerability

If you discover a potential security issue in this project, please report it by opening a private discussion or issue on the public repository, or by contacting the project owners directly. 

This project is a local-first offline CLI tool and does not connect to the internet, run a daemon, or expose network ports, which limits the potential attack surface.
